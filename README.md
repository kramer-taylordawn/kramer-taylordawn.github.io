# GitHub Markdown Reference

A complete copy-paste guide for GitHub Flavored Markdown (GFM).

---

## Headings
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

---

## Emphasis
- *Italic text*  
- **Bold text**  
- ***Bold and italic***  
- ~~Strikethrough~~

---

## Blockquote
> This is a blockquote.  
> It can span multiple lines.  
> > Nested blockquote

---

## Lists

### Unordered
- Item 1
- Item 2
  - Nested Item
    - Sub-nested Item
- Item 3

### Ordered
1. First item
2. Second item
   1. Sub item
   2. Another sub item
3. Third item

### Task List
- [x] Completed task  
- [ ] Incomplete task  
- [ ] Another pending task

---
